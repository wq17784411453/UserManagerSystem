# UserManagerSystem
<h2>系统简介</h2><br>  本管理系统针对于本校的羽毛球协会。用户有两种身份：一种是管理员，即相当于协会的会长和部长；另一种是普通用户，即协会干事。普通用户 具有按学号学好查询学生资料功能，普通用户必须先注册，然后登陆、查询。管理员 可以对会员（也是学生）进行添加学生资料、删除学生资料、修改学生资料、查询学生资料等操作。管理员必须先注册，然后登陆，进行一系列的操作。<br><br>
